# Welcome to the MOF Learning Handbook

This is a sample handbook written using **Jupyter Book + MyST Markdown**, focusing on **MOF (Metal–Organic Frameworks)**. It is available in both Chinese and English versions. [点此进入中文版本简介](intro-zh.md)

This handbook mainly demonstrates:

1. A MOF book sample compiled with MyST Markdown.
2. Specific writing templates, including:
   - Title and paragraph formatting
   - Various admonition boxes (warning, thought, exercise)
   - Inserting images and figure captions
   - Equations and references
   - Tables and lists
   - Code blocks and executable `code-cell`
   - Interactive plots/animations (output through Python)

:::{note}
You are free to modify the content and use it as a foundational writing template.
:::
